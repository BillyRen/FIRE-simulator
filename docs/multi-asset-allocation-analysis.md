# 长周期多资产配置研究：股债之外加入房地产 / 黄金

> 2026-05-31（**v4**:房产分"指数/个人房产"两情景 + 房产费 1.5% + 费率/波动敏感性 + **5% 网格** + **制度子样本分析**)。
> 参考 [`docs/four-asset-allocation-analysis.md`](four-asset-allocation-analysis.md) 的房产建模口径。
> 脚本 `analysis/multi_asset_allocation.py`,输出 `analysis/output/multi_asset_allocation/`。
> 方法对齐产品"最优资产配置"工具:Block Bootstrap + 覆盖率/成功率排名,资产含 Global Stock。
> **纯研究,未接入产品**。黄金数据见 [`docs/gold-data-preparation.md`](gold-data-preparation.md)。

## 0. 结论速览（TL;DR）

基准 = 三个金融资产(本国股 + 全球股 + 债)。提取 3.3% 实际、长周期、按**覆盖率/成功率**排名。房产按两种持有方式建模:

1. **房地产指数**(波动 1.0x,费 1.5%):最优房产配比 **45–80%**(池化 50%、美国 80%),覆盖率/P10 大幅提升。
2. **个人房产**(单套自住:波动 ×1.5、相关性稀释到 2/3、实际收益 floor 在 −100%,费 1.5%):最优房产配比**下移到 20–45%**(池化 35%、美国 45%),仍显著优于纯股债。
3. **黄金 = 强烈的制度依赖资产**(见 §6,核心洞察):金本位/战前期最优权重 **0**;**后金本位(1971+)最优权重飙到 25–30%**。黄金本质是"浮动汇率/法币时代"的对冲资产。
4. **中位终值的权衡(重要,不是"小幅")**:在高股市收益市场,按覆盖率最优会重仓房产 → **中位终值大幅下降**(美国 +指数房产 29M→**10M**、澳洲 46M→18M);池化市场则温和(23M→21M)。这是"用上行换下行保护"的真实代价。
5. **1.5% 费率**:见 §5——最优配比对费率**高度稳健**(0.5%→2.5% 全区间池化最优房产仅 40%→25%),1.5% 是合理中值但应按所在地调整。

### 其他单国（65 年,个人房产)
- **日本:** 股+债 81.2%(P10=0)→ 个人房产 10/65/0/25(89.5%);指数房产 0/45/0/55(94.8%)。房产是主要救星。

## 4. "如何变化"总结
| 情景 | 房产最优配比 | 黄金角色 |
|---|---|---|
| 房地产指数 | 45–80% | 多被房产挤出 |
| 个人房产(1.5x) | 20–45% | 补位增加(美/德/澳 5–20%) |

债券在全样本下几乎归 0(被房产替代);全球股是最优股票 sleeve。**但债券和黄金的角色高度依赖货币制度——见 §6。**

## 5. 关于 1.5% 费率（你的挑战 + 我的反驳)

### 5.1 费率敏感性（个人房产,65y,5% 网格)
| 费率 | 池化 最优房产 / 覆盖率 | 美国 最优房产 / 覆盖率 |
|---|---|---|
| 0.5% | 40% / 97.7% | 50% / 99.3% |
| 1.0% | 35% / 97.4% | 45% / 99.1% |
| **1.5%** | **35% / 97.0%** | **45% / 98.7%** |
| 2.0% | 30% / 96.6% | 40% / 98.4% |
| 2.5% | 25% / 96.3% | 35% / 98.0% |

**配比和定性结论对费率高度稳健:** 0.5%→2.5% 全区间,池化最优房产仅 40%→25%、覆盖率仅降 1.4pp;**即使按美国高房产税 2.5%,房产仍值 25–35% 并优于纯股债。** "1.5% 是否精确"对决策不关键。

### 5.2 我对 1.5% 的判断
1.5%(维护~1% + 税/物业~0.5%)是**发达市场自住房、长持有的合理中值,但依赖地区且可能略偏高:**
1. **地区差异大:** 美国仅房产税约 1.1%(部分州 2%+),自住总成本约 2.3–2.5%,1.5% 偏低;中国住宅历史上**无年度房产税**、维护低,总成本约 0.5–1%,1.5% 偏高;池化/全球 1.5% 是合理中值。
2. **JST 净/毛口径不确定性(可能重复计算):** 我们的 `Housing_Rent_YD` 是 rent/price(看似毛),但 JST 源序列在部分国家/年份可能已部分净化维护/折旧。若已净化,叠加 1.5% 维护会重复计算约 1%,有效附加成本应更接近 0.5–1%。无法从代码完全确认,标为不确定性。
3. **交易成本**(往返 5–10%)未计入,但长周期(40–65年)摊薄仅 ~0.1–0.2%/年,**支持**不必在持有成本外加太多。
4. **自住口径正确**(无空置/管理费);出租需再加 1–2%。

**建议用 §5.1 对号入座:** 中国自住房(无房产税)→ 看 0.5–1.0% → 房产 40–50%;美国自住房 → 看 2–2.5% → 房产 35–40%;全球中值 → 1.5% → 房产 35–45%。各档下"房产≥25%且优于债券"都成立。

### 5.3 波动倍数敏感性（费 1.5%,65y)
| 倍数 | 池化 房产/覆盖率 | 美国 房产/覆盖率 |
|---|---|---|
| 1.0x(指数) | 50% / 98.2% | 80% / 99.6% |
| **1.5x(基准)** | **35% / 97.0%** | **45% / 98.7%** |
| 2.0x | 25% / 96.2% | 30% / 98.1% |

波动倍数比费率更影响配比;即便 2.0x,房产仍值 25–30% 且优于纯股债基准。

## 6. ⭐ 制度子样本分析（不同回测时点,核心洞察)

把 Block Bootstrap 限制在不同历史窗口内重采样(池化 + 美国,个人房产 add_both,65y):

### 池化 ALL — 各制度最优配置 (DS/GS/DB/房/金)
| 制度窗口 | 基准 股+债 (成功率) | +两者 最优 (成功率) | 关键变化 |
|---|---|---|---|
| 全样本 1871+ | 20/80/0/0/0 (88.0%) | 5/60/0/35/0 (92.4%) | 房 35%,金 0 |
| 战前 1900–1945 | 20/80/0/0/0 (**66.7%**) | 10/70/0/20/0 (67.9%) | **乱世:加什么都几乎没用**;金 0 |
| 战后 1946+ | 30/70/0/0/0 (92.2%) | 10/35/0/35/**20** (**99.7%**) | 金跳到 **20%** |
| 后金本位 1971+ | 30/65/5/0/0 (90.5%) | 15/45/0/15/**25** (**99.9%**) | 金跳到 **25%**,房降 |
| 现代 1990+ | 15/40/45/0/0 (97.2%) | 5/30/**25**/10/**30** (**100%**) | 金 30% + **债回归 25%** |

### 各制度的资产实际年均收益（池化,%)
| 制度 | 本国股 | 全球股 | 债 | 房 | **金** |
|---|---|---|---|---|---|
| 战前 1900–45 | 5.1 | 8.1 | 0.6 | 5.0 | **0.8** |
| 战后 1946+ | 7.7 | 9.6 | 1.6 | 7.1 | **4.4** |
| 后金本位 1971+ | 8.1 | 8.3 | 3.0 | 6.0 | **6.4** |
| 现代 1990+ | 7.8 | 7.8 | 3.8 | 5.6 | **5.3** |

### 核心洞察
1. **黄金是"浮动汇率/法币时代"的资产,价值完全是制度性的。** 金本位/战前期金价被锚定 → 实际收益仅 0.8%、最优权重 **0**;1971 年金本位彻底破灭后,黄金实际收益升到 **6.4%** 并成为强力分散器 → 最优权重 **25–30%**。**是否配黄金,本质是在赌后 1971 的浮动法币体制(及黄金的货币对冲角色)能否延续。** 全样本均值(3.1%)掩盖了这个断裂——不要用它判断黄金。
2. **债券同样制度依赖:** 高通胀的战前/战后期债券实际收益仅 0.6–1.6%、几乎零配置;**1990+ 大缓和/反通胀期债券升到 3.8% 实际收益 → 最优配 25–45%**。债券不是"永远没用",而是"在反通胀周期才有用"。
3. **乱世(1900–1945)无解:** 含两次世界大战 + 大萧条,纯股债成功率仅 66.7%,加房产/黄金也只到 67.9%——**极端尾部下分散化的边际作用很小**。这是真正的"灾难情景"。
4. **房产跨制度最稳健**(多数制度 20–55% 最优),但乱世里也降到 20%。
5. **美国同构**:后金本位 1971+ 美国 +两者最优 = 60/0/0/15/**25**(25% 黄金、全球股归 0、房 15%);现代 1990+ = 60/0/10/5/**25**(25% 黄金)。

> ⚠️ 短窗口 + 65 年长周期 → block 环形重采样的复用率高(如 1990+ 仅 36 年填 65 年),近代窗口的 ~100% 成功率偏乐观;乱世窗口含最坏事件,其低成功率更可信。制度对比看**方向**而非绝对值。

## 7. 局限与重要警告
1. **覆盖率排名偏好"稳"而非"多"** → 重仓房产后中位终值在高股市市场大幅下降(美/澳 −50%~−66%),池化温和。若目标是期末财富最大化,结论不同。
2. **JST 房地产可投资性 + 净/毛口径**(§5.2)。指数口径假设能低成本持有分散房产篮子(现实难,除非非上市房产基金);上市 REITs **不适用**(与股市相关 0.6–0.7、波动近股票)。
3. **黄金制度依赖(§6)+ 与全球股共享 FX(0.53)**:全样本看黄金会低估其后 1971 价值、高估其全样本平均价值。
4. **3.3% 偏保守 → 只有长周期才区分配置**(用 40/65 年)。
5. **池化偏严苛;债券 `Domestic_Bond` 设定简化**(久期/再平衡更现实会改善债券角色)。
6. **制度子样本短窗口重采样**偏乐观(§6 警告)。

## 8. 对产品的潜在启示(非本次落地)
- **房地产**:UI 须区分**指数 / 自住单房 / REITs** 三种口径,允许用户设持有成本(默认 1.5%,按地区调);不要呈现指数口径的超高权重。
- **黄金**:不应给"全样本最优"的单一数字;其价值依赖货币制度,应作为小比例制度对冲卫星仓(后 1971 视角 10–25%),并提示制度依赖性。
- **债券**:在反通胀情景才有显著价值。
- 与 [[optimal-allocation-cross-strategy]](30/60/10)、[[guardrail-allocation-leverage-sweep]] 互补。
