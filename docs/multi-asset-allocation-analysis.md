# 长周期多资产配置研究：股债之外加入房地产 / 黄金

> 2026-05-31（v2,改用产品"最优资产配置"口径:Block Bootstrap + 成功率/覆盖率排名,含 Global Stock)。
> 脚本 `analysis/multi_asset_allocation.py`,输出 `analysis/output/multi_asset_allocation/{results.csv, asset_stats.csv}`。
> **纯研究分析,未接入产品**。黄金数据见 [`docs/gold-data-preparation.md`](gold-data-preparation.md)。

## 0. 结论速览（TL;DR）

基准 = 产品的三个金融资产(本国股 + 全球股 + 债券)。在 **3.3% 实际提取、0.5% 费用、长周期**下,按**成功率 / 覆盖率**排名:

1. **加入本国房地产 = 影响最大。** 抢占 40–90% 权重,**同时挤掉债券和本国股**(最优组合变成"全球股 + 房地产"),成功率/覆盖率大幅跳升,**P10 终值从≈0 拉到数百万**(关键的尾部保护)。
2. **单独加入黄金 = 小幅分散器。** 占 10–20%,覆盖率仅小幅提升(池化 +0.2~0.6pp);**在高通胀的德国权重最高(~20%)**,边际价值最大。注意黄金与**全球股**相关高达 0.53(共享 FX),只与本国股近零相关(0.02)——对以全球股为主的组合,黄金分散价值有限。
3. **同时加两者 = 房地产主导,黄金大多被挤出**(池化/英/日 黄金归 0;仅美/德/澳保留 ~10%)。二者作为分散器是部分替代。
4. **全球股 > 本国股**:几乎所有最优解都重仓全球股(与产品默认的全球化倾向一致)。

> ⚠️ **两个重要提醒**(详见 §6):(a) 成功率排名天然偏好"低收益但更稳"的房地产 → 最优解的**中位终值反而下降**(牺牲上行换下行保护);(b) **JST 房地产严重高估可投资性**(无杠杆/无成本/平滑波动/指数≠单套房)→ "40–90% 房地产"是**机械上界**,真实权重远低。方向稳健,幅度高估。

## 1. 方法（对齐产品"最优资产配置"工具)

| 维度 | 设定 |
|---|---|
| 资产域 | 本国股 `Domestic_Stock`、**全球股 `Global_Stock`**、债券 `Domestic_Bond`、房地产 `Housing_TR`、黄金(`jst_gold.csv`) |
| 菜单 | 基准 = 股+债(三金融资产);分别 +房地产 / +黄金(四资产)/ +两者(五资产) |
| 采样 | Block Bootstrap(复用产品引擎),所有资产同一组 block 索引联合采样保相关性;池化(sqrt-GDP 加权,= 产品 `country="ALL"`)+ 单国(美/日/英/德/澳);block 5–15,3000 路径 |
| 组合实际收益 | **产品口径**(`portfolio.py`):`名义组合 = Σ wₐ·(名义ₐ − 费用ₐ)`,再 `实际 = (1+名义组合)/(1+通胀) − 1`(先加权扣费、后除通胀一次)。组合层除通胀天然驯服超通胀年的 FX 尖刺,故**不 winsorize**(与产品一致) |
| 提取策略 | **固定策略**(产品 `_simulate_vectorized_fixed_from_matrix`):先增长后提取恒定实际额,资产 ≤0 判耗尽。初始 **100万**、年提 **33,000 实际(3.3%)**、费用 **0.5%/资产**、杠杆 1.0 |
| 排名指标 | **覆盖率/funded ratio**(平均覆盖比例)为主键 —— 与产品 allocation 页一致(`best = max(..., key=funded_ratio)`);**成功率**(覆盖全退休期占比)为 tiebreak。直接调用产品 `compute_funded_ratio`/`compute_success_rate` |
| 年限 | **40 与 65 年**(65 = 产品默认;3.3% 偏保守,只有长周期才能让成功率/覆盖率区分配置;30 年几乎全部成功) |

## 2. 资产特性（变化的根本驱动)

实际年收益(描述表为可读性按 [−95%,+300%] clip;**主分析用 raw**)。池化为 sqrt-GDP 加权:

| 实体 | 本国股 | 全球股 | 债券 | 房地产 | 黄金 |
|---|---|---|---|---|---|
| **池化 ALL** | 7.0% / 21.5% | 9.1% / 25.3% | 1.5% / 11.7% | **6.6% / 11.5%** | 3.1% / 23.2% |
| 美国 | 8.6% / 18.5% | 8.5% / 22.4% | 2.0% / 9.4% | 6.3% / 7.8% | 2.3% / 15.3% |
| 日本 | 6.6% / 24.7% | 10.7% / 38.0% | 0.1% / 14.3% | 9.6% / 17.8% | 4.3% / 35.7% |
| 德国 | 7.3% / 23.4% | 8.5% / 22.0% | 1.4% / 12.3% | 5.5% / 14.0% | 4.2% / 22.2% |
| 澳洲 | 8.3% / 15.4% | 8.7% / 17.7% | 1.6% / 8.3% | 5.9% / 11.2% | 2.2% / 14.2% |

相关性(池化,实际):本国股–全球股 **0.34**;本国股–债 0.31;股–房 **0.07–0.17**;**金–本国股 0.02 / 金–全球股 0.53**;房–金 ~0。

**关键:**
- **房地产**:股票般收益(6.6%≈本国股 7.0%)+ 约一半波动(11.5%)+ 与股票低相关(0.07–0.17)→ 风险调整后最强稳定器。
- **黄金**:实际收益低(3.1%)+ 与本国股近零相关(0.02)→ 对本国股是好分散;**但与全球股相关 0.53(共享 FX)**,故对以全球股为主的组合分散价值有限。
- **债券**:低收益 + 与股票正相关(0.32)→ 最弱的长周期分散器,一有房地产即被替代。

## 4. 最优配置"如何变化"总结

| 加入资产 | 最优配置怎么变 | 成功率/覆盖率/P10 |
|---|---|---|
| **本国房地产** | 抢 40–90%,**债券和本国股一起归零**,只留全球股 + 房地产 | 大幅提升;P10 终值从≈0 拉到数百万(尾部保护) |
| **黄金** | 占 10–20%(高通胀的德国最高 ~20%),股票仍重 | 小幅提升;德国边际最大,日本几乎无效 |
| **两者同时** | **房地产主导**,黄金被挤到 0–10%(仅美/德/澳留 ~10%) | ≈ 只加房地产;黄金边际贡献大多被覆盖 |

跨 40/65 年、跨国稳健。效果在**长周期 + 危机/高通胀市场**(日德、池化)最显著——那里基准的耗尽尾部最厚(P10=0)。

## 5. 与 v1(CRRA 口径)的一致性

本 v2(产品成功率/覆盖率口径 + 含全球股)与上一版(CRRA CE 口径、本国股)在**核心方向上完全一致**:房地产主导、黄金小分散器、两者同时则房地产挤出黄金。说明结论对方法选择稳健。差异:本版正确纳入全球股,显示最优股票 sleeve 偏全球;并用更易解释的成功率/覆盖率/P10 替代尾部敏感的 CRRA。

## 6. 局限与重要警告

1. **成功率排名 ≠ 财富最大化。** 它偏好"低波动可持续"的组合,因此最优解重仓房地产后**中位终值反而下降**(美国 65y 基准 29M → +房地产 16M)——用上行换下行保护(P10/成功率)。若你的目标是期末财富最大化而非可持续性,结论会不同。
2. **JST 房地产严重高估可投资性(最关键)。** 无杠杆、未扣交易/维护/管理/空置成本(仅含租金毛收益)、年度+评估平滑**低估真实波动**、且全国指数 ≠ 你持有的单套房(巨大特异性 + 流动性风险)。故 40–90% 是**机械上界**,真实可投资权重远低。**可信的是方向:房地产应占一块有意义配置、能分散股票、优于债券——不是"砸 80% 进房子"。** 实务可考虑用 REIT 替代。
3. **黄金依赖制度且与全球股共享 FX。** 金本位期(1871–1971)实际收益≈0,浮动期才有分散价值;FX 暴露对非美元投资者真实有意(货币贬值对冲),但也使黄金与全球股相关高达 0.53,分散全球股的能力远弱于分散本国股(0.02)。
4. **3.3% 偏保守 → 只有长周期才区分配置。** 30 年内几乎全部成功,排名无意义;故用 40/65 年。
5. **全球股 raw FX 离群值:** 主分析用产品口径(组合先加权后除通胀一次),单年超通胀 FX 尖刺被通胀分母自然驯服,且与产品行为一致;描述表(§2)为可读性按 [−95%,+300%] clip。
6. **池化偏严苛 + 债券设定简化。** 池化代表"全球平均投资者"(含日德灾难),其绝对成功率低于美国中心口径,只看**菜单间相对变化**;债券用 `Domestic_Bond`,更现实的久期/再平衡会改善债券角色但仍难敌房地产。

## 7. 对产品的潜在启示(非本次落地)

- **房地产**:作为"股票低相关增益资产"暴露最有价值,但 UI 必须警告可投资性(成本、流动性、单房特异性、建议 REIT 口径),且不要直接呈现机械最优的超高权重。
- **黄金**:定位为"高通胀/危机尾部对冲的小比例卫星仓(~10%)",默认值应随用户本币稳定性/通胀暴露调整;对全球股为主的组合,黄金的分散增益有限。
- 与 [[optimal-allocation-cross-strategy]](baseline 30/60/10)、[[guardrail-allocation-leverage-sweep]] 互补:那些在"股+债(含全球股)"内优化,本研究回答"再加一类资产时边界怎么动"。
