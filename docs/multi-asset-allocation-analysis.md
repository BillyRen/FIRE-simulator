# 长周期多资产配置研究：股债之外加入房地产 / 黄金

> 2026-05-31（**v3**:房产分"指数 / 个人房产"两情景 + 房产费 1.5% + 费率/波动敏感性。
> 参考 [`docs/four-asset-allocation-analysis.md`](four-asset-allocation-analysis.md) 的房产建模口径)。
> 脚本 `analysis/multi_asset_allocation.py`,输出 `analysis/output/multi_asset_allocation/`。
> 方法对齐产品"最优资产配置"工具:Block Bootstrap + 成功率/覆盖率排名,资产含 Global Stock。
> **纯研究,未接入产品**。黄金数据见 [`docs/gold-data-preparation.md`](gold-data-preparation.md)。

## 0. 结论速览（TL;DR）

基准 = 产品的三个金融资产(本国股 + 全球股 + 债)。提取 3.3% 实际、长周期、按**覆盖率/成功率**排名。房产按两种持有方式分别建模:

1. **房地产指数**(分散的全国住宅指数,波动 1.0x,费 1.5%):最优房产配比 **40–80%**(池化 50%、美国 80%),覆盖率/P10 大幅提升。
2. **个人房产**(单套自住,波动 ×1.5、相关性稀释到 2/3,费 1.5%):非系统性风险拖累下,最优房产配比**下移到 20–40%**(池化 30%、美国 40%),但仍显著优于纯股债。
3. **黄金**:小分散器,占 10–20%(高通胀德国最高);**当房产是"个人房产"(更差)时,黄金在 +两者里反而补位更多**(美/德/澳 +both 留 10–20% 黄金)。
4. **关于 1.5% 费率(回应你的挑战)**:见 §5——结论是**配比对费率高度稳健**,1.5% 是合理中值但应按你所在地调整;**即使按美国高房产税 2.5%,房产仍值 30% 且优于基准**。

> 与参考文档一致:个人房产把最优房产配比从指数的 ~40-50% 下移到 ~25-40%。两种方式下加房地产都显著改善尾部(P10 从≈0 升到数百万),中位终值小幅下降(用上行换下行)。

### 其他单国（65 年,个人房产口径)

- **日本(危机型):** 股+债 81.2%(P10=0)→ 指数房产 0/50/0/50(94.6%)→ 个人房产 10/70/0/20(87.8%,房产降到 20%)。房产仍是主要救星,黄金几乎无效。
- **德国(高通胀):** 个人房产 +两者保留 **20% 黄金**(0/60/0/20/20,93.4%)——房产变差时黄金补位最明显。
- **英国:** 指数房产 50% → 个人房产 30%;+两者个人含 10% 黄金。
- **澳洲(高且稳股市):** 基准已 97.5%;指数房产 50% → 个人房产 30%,+两者个人含 20% 黄金。

**核心观察:** 个人房产口径下,(a) 房产最优配比普遍**腰斩到 20–40%**;(b) 让出的份额回流到全球股,以及——尤其在美/德/澳——**回流到黄金**(黄金与个人房产互补性强于与指数房产)。

## 4. 最优配置"如何变化"总结

| 情景 | 房产最优配比 | 黄金角色 | 机制 |
|---|---|---|---|
| **房地产指数** | 40–80% | 多被房产挤出(0–10%) | 房产低波动(11.5%)、低股相关 → 极强稳定器 |
| **个人房产(1.5x 波动)** | 20–40% | **补位增加**(美/德/澳 10–20%) | 非系统性风险拖累房产,黄金的零相关分散价值相对上升 |

债券在两种情景下几乎都归 0(被房产替代)。全球股是最优股票 sleeve。

## 5. 关于 1.5% 费率是否合理（你的挑战 + 我的反驳)

### 5.1 敏感性:费率 0.5%–2.5%（个人房产,65 年)

| 费率 | 池化 最优房产 / 覆盖率 | 美国 最优房产 / 覆盖率 |
|---|---|---|
| 0.5% | 40% / 97.6% | 50% / 99.3% |
| 1.0% | 40% / 97.3% | 50% / 99.1% |
| **1.5%** | **30% / 96.9%** | **40% / 98.7%** |
| 2.0% | 30% / 96.6% | 40% / 98.3% |
| 2.5% | 30% / 96.3% | 30% / 97.9% |

**关键结论:配比和定性结论对费率高度稳健。** 费率横跨 0.5%→2.5% 整个合理区间,池化最优房产仅从 40% 降到 30%、覆盖率仅降 1.3pp;**即使按美国高房产税 2.5%,房产仍值 30% 并优于纯股债基准。** 所以"1.5% 是否精确"对决策远没那么关键。

### 5.2 我对 1.5% 的判断(挑战你的思考)

**1.5% 是发达市场自住房、长持有的合理中值,但它依赖地区,而且可能略偏高:**

1. **地区差异大,1.5% 是中值而非普适值:**
   - **美国**:仅房产税就约 1.1%(全国均值,部分州 2%+),加维护 1% + 保险 0.3% → 自住总持有成本约 **2.3–2.5%**,1.5% 偏低。
   - **中国(你很可能的基地)**:住宅历史上**无年度房产税**、维护成本低 → 总持有成本约 **0.5–1.0%**,1.5% 偏高。
   - 池化/全球投资者:1.5% 是合理中值。
2. **JST 数据的净/毛口径不确定性(可能导致重复计算):** 我们的 `Housing_Rent_YD` 是 rent/price(看似**毛**租金收益率),但 JST 源序列在部分国家/年份可能已部分净化维护/折旧。若已净化,再叠加 1.5% 维护就**重复计算约 1%**,则有效附加成本应更接近 **0.5–1%**。我无法从代码完全确认 JST 内部口径,标为不确定性——这是 1.5% 可能偏高的第二个理由。
3. **交易成本未计入,但长周期可忽略:** 买卖往返 5–10% 在 40–65 年里摊薄后仅 ~0.1–0.2%/年。这反而**支持**长周期下不必在持有成本外加太多。
4. **自住 vs 出租口径正确:** 1.5% 是自住口径(无空置/管理费);出租投资者需再加 1–2%。你说的是自住房,框定正确。

**我的建议:** 与其纠结单一 1.5%,不如**用 §5.1 敏感性表对号入座**:
- **中国自住房**(无房产税)→ 看 0.5–1.0% 行 → 最优房产 **40–50%**;
- **美国自住房**(高房产税)→ 看 2.0–2.5% 行 → 最优房产 **30–40%**;
- **全球/池化中值** → 1.5% 行 → 最优房产 **30–40%**。

无论哪一档,"房地产应占一块有意义配置(≥30%)、优于债券"的结论都成立。

### 5.3 个人房产波动倍数敏感性（费率 1.5%,65 年)

| 波动倍数 | 池化 最优房产 / 覆盖率 | 美国 最优房产 / 覆盖率 |
|---|---|---|
| 1.0x(指数) | 50% / 98.2% | 80% / 99.6% |
| 1.25x | 40% / 97.5% | 50% / 99.1% |
| **1.5x(基准假设)** | **30% / 96.9%** | **40% / 98.7%** |
| 1.75x | 30% / 96.5% | 40% / 98.4% |
| 2.0x | 20% / 96.1% | 30% / 98.1% |

波动倍数对最优配比的影响**大于**费率:从指数(1.0x)到 2.0x,池化最优房产从 50% 降到 20%。即便 2.0x,房产仍值 20% 且覆盖率(96.1%)高于纯股债基准(94.8%)。

## 6. 局限与重要警告

1. **成功率/覆盖率排名偏好"稳"而非"多"** → 重仓房产后中位终值下降(用上行换下行)。若目标是期末财富最大化,结论不同。
2. **JST 房地产可投资性 + 净/毛口径**(见 §5.2)。指数口径假设你能低成本持有分散的全国房产篮子(现实中很难,除非非上市房产基金);个人房产口径更贴近单套自住房。上市 REITs **不适用本研究**(REITs 与股市相关 0.6–0.7、波动近股票,分散价值远低)。
3. **黄金依赖制度且与全球股共享 FX(0.53):** 金本位期实际收益≈0;对全球股为主的组合,黄金分散有限,故权重小。
4. **3.3% 偏保守 → 只有长周期才区分配置**(用 40/65 年)。
5. **池化偏严苛 + 债券设定简化:** 只看菜单间相对变化;`Domestic_Bond` 久期/再平衡更现实会改善债券角色但仍难敌房产。
6. **全球股 raw FX 离群值:** 主分析用产品口径(组合层除通胀)自然驯服;描述表(§2)按 [−95%,+300%] clip。

## 7. 对产品的潜在启示(非本次落地)

- **房地产**:作为"股票低相关增益资产",但 UI 必须区分**指数 / 自住单房 / REITs** 三种口径(波动与相关性差异巨大),并允许用户设持有成本(默认 1.5%,可按地区调)。不要直接呈现指数口径的超高权重。
- **黄金**:小比例卫星仓(~10%),对个人房产组合的补位价值高于对指数房产组合。
