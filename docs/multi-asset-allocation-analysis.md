# 长周期多资产配置研究：股债之外加入房地产 / 黄金

> 2026-05-31。脚本 `analysis/multi_asset_allocation.py`，输出
> `analysis/output/multi_asset_allocation/{results.csv, asset_stats.csv}`。
> **纯研究分析,未接入产品**。黄金数据见 [`docs/gold-data-preparation.md`](gold-data-preparation.md)。

## 0. 结论速览（TL;DR）

从"只有股票+债券"出发,在长周期(30/40 年)实际收益框架下:

1. **加入本国房地产 = 影响最大。** 它会在最优组合里占到很大权重(JST 池化下 γ2 最优 ~80%、美国 ~50%),把组合波动**砍掉约一半**,并**完全替代债券**——房地产在这套数据里是比债券强得多的"稳定器"(股票般的收益 + 债券般甚至更低的波动 + 与股票低相关)。
2. **单独加入黄金 = 温和的分散器。** 占比适中(池化 ~30%、美国 ~10%),小幅降波动、小幅提升风险调整后收益;**在高通胀/危机型市场(德国、日本)权重更高**,在股票收益高且稳的市场(澳洲)几乎为 0。
3. **同时加入两者 = 房地产主导,黄金大多被挤出。** 房地产已经提供了分散收益,黄金只在通胀创伤最深的德国(+both 仍保留 ~10% 黄金)还有边际价值。**房地产与黄金是部分替代关系**。

> ⚠️ **最重要的警告(详见 §6):JST 房地产收益严重高估了"可投资性"**——无杠杆、不含交易/维护/空置成本、年度数据平滑低估真实波动、且全国房价指数 ≠ 你实际持有的单套房(巨大特异性风险)。所以"40-100% 房地产"是**机械上界**,真实可投资权重远低于此。**方向(房地产值得一块有意义的配置、能分散股票、优于债券)稳健;幅度被高估。**

## 1. 方法与数据

| 维度 | 设定 |
|---|---|
| 资产域(实际收益) | 本国股票 `domestic_stock`、本国债券 `domestic_bond`、房地产 `Housing_TR`、黄金(`jst_gold.csv`) |
| 基准菜单 | 股+债;然后分别 +房地产 / +黄金 / +两者 |
| 数据 | JST 1871–2025,16 国;池化(ALL,sqrt-GDP 加权,与产品 `country="ALL"` 一致)+ 单国(美/日/英/德/澳) |
| 采样 | Joint block bootstrap(复用产品引擎),所有资产用**同一组 block 索引**联合采样以保留跨资产相关性;5000 路径,block 5–15 年 |
| 共同样本 | 每国仅取四资产 + 通胀**全部非空**的年份,使四个菜单跑在**同一时间窗**(隔离"加资产"效应,而非"换窗口"效应) |
| 稳健化 | 每国对各资产实际收益按 **[1,99] 分位 winsorize**,压制超级通胀年 FX/CPI 错位的单年测量尖刺 |
| 排名指标 | CRRA 确定性等价(γ=1/2/4/8)、年化实际波动、奖励/波动、P10 期末财富;**headline 用 γ=2(中等风险厌恶)**;另跑提取阶段视角:90% 成功率下的最大固定实际提取率(SWR90)、4% 提取的成功率 |

**两个关键方法学选择(都已在脚本 docstring 记录):**

- **股票用本国股而非 `Global_Stock`。** 产品的 `Global_Stock` 是 GDP 加权世界股票经 FX 换算成本币;在池化/日德战后超级通胀样本里,它和黄金共享同一汇率冲击,产生单年 >100% 波动和 ~0.9 的伪相关,严重扭曲矩估计。本国股是本币、无 FX 换算,干净,也是 "rate of return on everything" 文献的标准做法。黄金保留其 FX 暴露(对非美元投资者,货币贬值对冲是黄金收益的真实组成)。
- **γ=2 做 headline,γ=4 仅作"下行规避方向"对照。** γ=4 的 CRRA CE 被极少数期末财富趋零的路径主导(`E[W^{-3}]` 爆炸),其*幅度*和它选出的高黄金权重是尾部假象,不能当推荐。

## 2. 资产特性(变化的根本驱动)

实际年收益,winsorized(完整见 `asset_stats.csv`):

实体行中 **池化 ALL 为 sqrt-GDP 加权**(与 bootstrap 抽样总体一致;每行权重 = 国家 GDP 权重 / 该国年数),单国为等权。

| 实体 | 股票 均值/波动 | 债券 | 房地产 | 黄金 |
|---|---|---|---|---|
| **池化 ALL** | 6.9% / 20.3% | 1.5% / 10.8% | **6.6% / 10.3%** | 2.5% / 17.7% |
| 美国 | 8.6% / 18.1% | 2.0% / 9.3% | 6.2% / 6.6% | 2.1% / 14.3% |
| 日本 | 6.8% / 23.4% | 0.4% / 12.4% | 10.0% / 15.0% | 2.9% / 23.0% |
| 德国 | 6.9% / 21.4% | 1.6% / 9.5% | 5.8% / 11.7% | 3.7% / 18.6% |
| 澳洲 | 8.3% / 15.0% | 1.5% / 7.9% | 5.3% / 7.0% | 2.2% / 13.9% |

实际收益相关性(池化):

| | 股 | 债 | 房 | 金 |
|---|---|---|---|---|
| 股 | 1 | 0.32 | 0.16 | 0.05 |
| 债 | | 1 | 0.15 | 0.03 |
| 房 | | | 1 | 0.01 |
| 金 | | | | 1 |

**两点决定了一切:**
- **房地产 = 股票般的收益(6.6%≈股 6.9%) + 一半的波动(10.3%) + 与股票低相关(0.16)** → 风险调整后碾压。这正是 Jordà 等 "rate of return on everything" 的核心发现。
- **黄金 = 低实际收益(2.5%) + 与股票几乎零相关(0.05,美国 -0.03)** → 价值纯在分散,因此权重适中。
- 债券在四者里是最弱的长周期分散器:低收益 + 与股票正相关(0.32)。

### 其他单国要点

- **日本(危机型):** 纯股债灾难(CE₂ −1.73%、SWR90 仅 **1.0%**,战后股灾尾部)。加房地产直接救场(0/0/**100**/0,SWR90 5.25%)。黄金单独加只能把股票尾部部分对冲(50% 金)但因低收益 SWR 仍仅 ~1.8%——**房地产是救星,黄金只是逃命工具**。
- **德国(高通胀):** 唯一一个 +两者仍**保留黄金(50/0/40/**10**)** 的市场——德国恶性通胀史使黄金的尾部对冲在房地产之外仍有边际价值。单独 +黄金权重也最高(30%)。
- **英国:** +房地产 40/0/60,波动 16.8%→8.8%;+黄金仅 10%。
- **澳洲(高且稳的股票收益):** 股票太强,+黄金最优权重 **0%**;+房地产也只要 20%。说明黄金/房地产的价值依赖股票本身有多"不够好"。

## 4. 跨风险厌恶 / 跨年限的稳健性

- **方向在 γ=1→8、30y/40y、所有实体上一致:** 加房地产 > 加黄金;债券被房地产替代;同时加则房地产主导。
- **风险厌恶越高 → 房地产/黄金权重越大、股票越小**(γ4↓ 列):因为两者都压缩灾难性左尾。但 γ4 的黄金权重(池化 60%、德国 40%)是尾部假象主导,不作推荐。
- **黄金权重随"通胀/危机暴露"上升:** 澳洲 0% < 美/英 10% < 池化/德国 30% < 日本(逃命)50%。黄金是**对冲品而非核心持仓**。

## 5. 对最优配置"如何变化"的总结

| 加入的资产 | 最优配置怎么变 | 机制 |
|---|---|---|
| **本国房地产** | 抢占 40–100% 权重,**债券归零**,股票降一截;波动腰斩,SWR / 成功率 / CE 全面跳升 | 股票般收益 + 低波动 + 低股相关,风险调整后优于股和债 |
| **黄金** | 占 10–30%(通胀/危机市场更高),股票略降,债券仍≈0;小幅降波动、提 CE | 实际收益低但与股票零相关,纯分散 + 通胀尾部对冲 |
| **两者同时** | **房地产主导**,黄金被挤到 0–10%(仅德国保留 ~10%) | 房地产已提供大部分分散收益,黄金边际价值大多被覆盖,二者部分替代 |

## 6. 局限与重要警告

1. **房地产可投资性被严重高估(最关键)。** JST `Housing_TR` 是:无杠杆、未扣交易/维护/管理/空置成本(仅含租金毛收益)、年度且含评估平滑(**低估真实波动**)、且是**全国房价指数**——而你实际持有的是单套房,有指数无法捕捉的巨大特异性风险与流动性折价。因此机械最优的 40–100% 房地产是**上界**;真实可投资权重(经成本/杠杆/特异性调整后)显著更低。**可信的结论是方向:房地产应有一块有意义配置、能分散股票、长周期优于债券——不是"把 80% 砸进房子"。**
2. **黄金的统计画像依赖样本制度。** 1871–1971 金本位/布雷顿森林期黄金实际收益≈0(名义近固定),浮动期(1971+)才有高波动与正实际收益。Block bootstrap 会保留这种制度突变,黄金的有效贡献高度取决于采样窗口落在哪个制度。FX 暴露对非美元投资者是真实且有意的(货币贬值对冲)。
3. **γ=4 CE 幅度是尾部假象。** 仅用于判断"方向",不可读其数值或其选出的高黄金权重。
4. **方法学取舍已记录:** 剔除 `Global_Stock`(池化 FX 伪影)、winsorize [1,99]、共同样本对齐。换不同设定数值会变,但 §3/§5 的方向稳健。
5. **池化 SWR 偏严苛。** 池化代表"全球平均投资者"经历(含日德战后等灾难),其 SWR 绝对值低于以美国为中心的口径,不要直接和产品默认值比较;此处只看**菜单之间的相对变化**。
6. **债券设定简化。** 用 `Domestic_Bond`,长周期实际收益低;若用更现实的久期/再平衡策略,债券角色会改善,但仍难敌房地产的风险调整后表现。

## 7. 对产品的潜在启示(非本次落地)

- 若未来把房地产/黄金做成可选资产类别:**房地产**应作为"股票的低相关增益资产"暴露,但 UI 需对可投资性警告(成本、流动性、单房特异性、是否用 REIT 替代);**黄金**定位为"通胀/危机尾部对冲的小比例卫星仓(~10%)",且默认值应随用户的通胀暴露/本币稳定性调整。
- 与现有研究衔接:本结论与 [[optimal-allocation-cross-strategy]](baseline 30/60/10)、[[guardrail-allocation-leverage-sweep]] 互补——那些在"股+债(含全球股)"框架内优化,本研究回答"再加一类资产时边界怎么动"。
