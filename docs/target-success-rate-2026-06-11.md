# 目标成功率研究：固定提取 vs 风险护栏（用户真实场景，2026-06-11）

**问题**：用户（35 岁，家庭=夫妻+1 孩，¥24M，65 年退休期，全套概率现金流）
(1) 固定提取策略下目标成功率定多少；(2) 风险护栏策略下，按**最终 realized 成功率**口径目标定多少。

计划：`docs/plan-2026-06-11-target-success-rate.md`（Codex 评审 22 findings 已吸收）
脚本：`analysis/target_success_rate_2026_06_11.py`；输出 `analysis/output/target_success/*.csv`
设置：33/67/0、expense 0.005、block 5-15、N=20,000（固定）/10,000（护栏）、seeds {42, 60042, 120042}、
向量化循环与 `run_simulation_from_matrix` 逐位等价（脚本内 assert）。

## 1. 口径

| 口径 | 含义 |
|---|---|
| raw success | 65 年内资产从未耗尽（`compute_success_rate`） |
| eff_rel50 | 未破产 **且** 基础消费未跌破初始值一半 |
| eff_abs30w | 未破产 **且** 基础消费未跌破绝对 ¥30 万线（敏感性 25/35 万） |
| severe_fail | funded ratio < 0.5（前 32.5 年内破产） |
| CEW | CRRA γ=2、δ=2% 的确定性等价年消费（基础消费口径，期望支出 proxy 还原） |
| 在世破产 | Gompertz(SSA 2021) 加权 P(破产时夫妻至少一人在世) |

数据源：**POOL**（JST 16 国等概率 1900+，用户 baseline）、**JST_USA**（1900+ 长史）、**FIRE_US**（1970+ 现代）。
注意 FIRE_US 并非最乐观源——用户 67% 配置在 global stock，1970+ 的 EAFE（含日本失落 30 年）拖累该源，
其成功率与 POOL 接近而非 JST_USA。

## 2. 固定提取结果

### 2.1 各目标成功率对应的最大可持续基础年消费（万/年，3-seed 均值，seed 间差 ≤1.5 万）

| 目标成功率 | POOL | JST_USA | FIRE_US |
|---|---|---|---|
| 75% | 69.4 | 79.1 | 62.7 |
| 80% | 60.1 | 71.5 | 57.4 |
| **85%** | **50.3** | 63.2 | 51.7 |
| **90%** | **38.8** | 53.0 | 45.2 |
| 95% | 24.8 | 38.8 | 36.4 |
| 97.5% | <20（网格外） | 26.6 | 29.6 |
| 99% | <20 | <20 | 22.4 |

- **用户当前 ¥50 万 = POOL 85.1% / JST_USA 91.3% / FIRE_US 86.3%**（Wilson 95% CI ±0.5pp）。
- 边际代价递增（POOL）：80→85% 砍 9.9 万/年；85→90% 砍 11.5 万；90→95% 砍 **14.0 万**。
- **POOL + 65 年下 97.5%+ 不可达（基础消费 <20 万）**——"接近绝对安全"在全球数据 65 年口径下不存在，
  与 Morningstar"100% 目标把 30y SWR 压到 2.5%"和 ERN 60y 结论方向一致但更极端。
- fragile success（成功但终值 <3 年支出）仅 0.2-0.5%——成功路径几乎不擦边，65 年期成败分明。

### 2.2 失败时间结构：死亡率折扣对 35 岁退休者失效

- 破产年份中位 = 第 28-31 年（用户 63-66 岁）；@50 万 有 5.8% 概率在 25 年内（社保开始前）破产。
- @50 万：模型失败 14.9%，其中**夫妻至少一人在世时破产 = 13.4%**——死亡率折扣只有 ~10%。
- 经典论点"长 horizon 的模型失败多发生在人已离世的尾部"（Kitces/Milevsky，对 65 岁退休成立）
  **对 35 岁退休不成立**：失败集中在 60-70 岁，存活概率仍 >90%。模型内成功率不能靠死亡率打折。

### 2.3 稳健性

- block 5-15 vs 8-20（POOL）：成功率 +0.9~1.9pp——小于数据源间差异（同消费下 POOL vs JST_USA 差 6pp）。
- seed 漂移：反解消费 ≤1.5 万；成功率 ≤0.4pp。

## 3. 风险护栏结果（F 族：upper=0.99 / adj=0.05 / amount / mr=1 / lower=target−0.10）

### 3.1 target → realized 映射（POOL，seed42；N=10,000；3-seed 漂移 raw ≤0.3pp）

| target | 反解初始消费 | raw realized | eff_rel50 | eff_abs30w | severe_fail | CEW 中位 | 中位年数<30万 | 在世破产 |
|---|---|---|---|---|---|---|---|---|
| 0.70 | 77.0 | 76.3% | 69% | 73% | 12.7% | 94.6 | 0 | 19.7% |
| 0.75 | 68.5 | 81.2% | 71% | 74% | 9.8% | 88.9 | 0 | 15.5% |
| 0.80 | 58.8 | 86.0% | 63% | 61% | 6.8% | 81.2 | 0 | 11.3% |
| **0.85** | **49.9** | **89.8%** | 65% | **61%** | **4.9%** | **72.5** | 0 | 8.2% |
| 0.90 | 37.4 | 93.7% | 68% | 43% | 3.0% | 60.3 | 4 | 4.9% |
| 0.95 | 23.5 | 96.6% | 67% | **0%** | 1.6% | 41.9 | **14** | 2.8% |

插值：**realized 90% ⇔ target≈0.85 ⇔ 初始消费 ≈49-50 万**；realized 95% ⇔ target≈0.92 ⇔ 初始 31 万。
对照源：JST_USA realized 90% ⇔ target≈0.82（init ~67 万）；FIRE_US ⇔ target≈0.78（init ~59 万）。
65y 映射与 50y 研究（target 0.85→realized 90.3%，[[project-allocation-cew-decision]]）基本一致。

### 3.2 三个结构性发现

1. **高 target 悖论**：target=0.95 反解初始消费 23.5 万已低于 30 万绝对生活线（eff_abs30w=0%），
   中位路径有 14 年生活在 30 万以下——**名义安全换实际贫困**。realized 目标不能贪高。
2. **eff_rel50 跨 target 几乎平坦（63-71%）**：相对 floor 随初始消费同比缩放 + adj=5% 累积削减动力学
   相同，导致该口径无区分度。**选 target 必须用绝对生活水准口径 + severe_fail + CEW**，相对口径会误导。
3. **CEW 中位 > 初始消费**（如 0.85 档 72.5 万 vs 49.9 万）：65 年长跑中位路径多次触发 upper=0.99 上调。
   护栏不只是"下行保护"，长 horizon 下也是"上行收割"——这是固定提取没有的期权价值。

### 3.3 同起点对比（¥50 万初始消费，POOL）

| 策略 | 成功率 | 中位生命周期消费 | 失败前形态 |
|---|---|---|---|
| 固定 50 万不调 | 85.1% | 50 万（恒定） | 无预警，破产即断崖 |
| 护栏 target=0.85 | **89.8%** | **CEW 72.5 万** | 多年渐进削减，有预警有回升 |

护栏的调整能力在同样初始生活水平下 **+4.7pp 成功率 + 显著更高中位消费**——两头都赢，代价是路径内
波动（约 65% 概率某年消费比初值低一半以上，多数随市场回升）。

## 5. 结论

**决策变量是消费水平，三个数据源是同一消费水平的三个成功率读数**（不为每个模型单设目标）。

### 问题 1：固定提取目标成功率 → **90%（POOL 口径，保守基准模型）**

- 即基础年消费 ≈ **39 万**；同一消费在 JST_USA ≈95%、FIRE_US ≈93%——三源全部 ≥90% 的稳健水平。
  POOL 是显式选择的保守基准（含 16 国灾难史），不是唯一真值——三源同表呈现即模型风险的量化。
- 不选 95%（POOL）：主因是**边际消费成本失衡**——要砍到 24.8 万（−14 万/年）买 5pp 模型内改善；
  辅证是模型间不确定性（同消费三源差 6-9pp）使模型内高位精度的意义有限。99% 在 POOL 下不可达。
- 不选 85% 以下作为严格固定的目标：85% 意味着 13.4% 在世破产、5.8% 在孩子大学毕业前破产，
  死亡率折扣失效（§2.2）且固定策略按定义无自救机制——对单收入家庭过高。
- **50 万消费水平只有两种一致的解读**（Codex 修正：不存在"固定但承诺削减"的中间态）：
  (a) 真·从不调整 → 接受 POOL 85.1%；(b) 愿意系统性削减 → 那就是护栏策略，恰好同一起点（见问题 2）。
  **50 万本身不是问题，配什么策略才是。**

### 问题 2：护栏 realized 成功率目标 → **raw realized ≈90%（POOL）+ 监控条件**

- **推荐参数：target=0.85 / lower=0.75 / upper=0.99 / adj=0.05 / amount / mr=1**
  → realized ≈ **90%**（3-seed 89.8-90.1%），反解初始基础消费 ≈ **50 万 = 用户当前水平正好可持续**。
- 监控条件（防"名义成功"操纵，非已稳健通过的硬 gate）：
  **severe_fail ≈5% 设计目标**——实测 4.4-4.9%（3 seeds，均值 4.6%），在目标内但无安全余量；
  **eff_abs30w（≈61%）作各 target 间对比指标**——它有区分度（0.90 档跌到 43%、0.95 档归零），
  但 61% 本身不构成"30 万底线被守住"的证明。
- **若 30 万是不可跌破的硬底线**：当前 amount 模式护栏不保证守住（参数上也无法保证）。
  正解是结构性的——把底线对应资金 ring-fence（低波动资产/年金化）或等待 floor-aware 双层提款规则
  （产品 backlog），而不是抬高 target。
- 不追 realized 95%：初始消费砍到 31 万、CEW −30%、绝对生活水准崩塌（§3.2 悖论）；
  realized <85%（target≤0.80）：severe_fail >6.6%、在世破产 >11%，超出家庭可承受。
- 65y 下 target→realized 映射 +4~5pp 与 50y 研究一致，结论可跨 horizon 沿用。

### 策略选择

同样 50 万起点：固定=85.1%、护栏=89.8% 且中位生命周期消费高 45%。**前提是真的能执行削减**——
约 35% 路径会经历"基础消费低于初值一半"的时期（含 10pp 破产路径；多数随市场回升），
这是护栏的体验预期，签约前必须接受。能接受 → 选护栏；不能接受任何削减 → 回到固定 90%/39 万。
固定提取的 90%/39 万同时用作规划基准锚。

## 6. 局限（Codex 计划评审采纳的 caveats）

1. 模型风险：Block Bootstrap 平稳性假设、JST 数据代表性——三源同表 + block 敏感性部分缓解，
   不可消除；POOL 在 walk-forward 验证中动荡期偏保守（[[project-walk-forward-mc-vs-backtest]]）。
2. 税、汇率换算成本、资本管制未显式建模（POOL 的多国灾难史是其粗代理）。
3. LTC 尾部风险未建模——维持 [[retirement-utility-considerations]] 的 ring-fence 10-15% 建议，
   该部分不参与上述 SWR/护栏计算。
4. CEW/floor 的基础消费口径用期望支出 proxy 还原（per-path 误差 ±几万，跨 target 排序不受影响）。
   **注意 proxy 只适用于分布统计**：路径内事件检测指标 `p_first_cut_by_15` 被 proxy 污染
   （"公办"路径实际支出比期望低 7.5 万/年即被误判为下调），已弃用，不进入结论。
5. 护栏 lookup table 用期望现金流 schedule（产品语义）——分支感知表可能改变高支出分支下的调整时点，
   但偏差已被 target→realized 实测映射吸收。
6. 死亡率用 SSA 2021（美国）：对中国居民寿命略偏长 = 破产风险方向上保守。
7. 结论条件于 33/67/0 附近配置（v2 研究表明护栏参数对 allocation 不敏感）。

### 7.2 批判与消化（6 条，4 条采纳修正、2 条部分采纳）

| Codex 批判 | 处理 |
|---|---|
| "50 万固定+承诺削减"在策略定义上不干净——承诺削减即条件策略，不应用固定口径描述 | **采纳**，§5 改为"两种一致解读" |
| severe_fail 4.9% 对 5% 阈值无安全余量（3-seed 漂移已 >0.1pp） | **采纳**，实测 4.4-4.9%，降级为监控条件 |
| eff_abs30w=61% 不能证明 30 万底线被守住；若是硬底线应进提款规则而非事后指标 | **采纳**，降级为对比指标 + 增加 ring-fence/floor-aware 正解 |
| "95% 是伪精度"过强——更准确是边际消费成本失衡（95% 与 90% 同样来自模型） | **采纳**，主因改为成本失衡、模型不确定性作辅证 |
| POOL 应明确为保守基准而非唯一真值 | 部分采纳（原意如此，措辞强化） |
| "应选护栏"缺行为前提——eff_rel50=65% 说明它是"高概率不破产但可能深度削减" | **采纳**，§5 增加体验预期与前提 |

### 7.3 Codex 增量策略建议（超出原问题范围，记录为后续方向）

1. **社保桥接期单独约束**：对 yr1-25（高支出+无社保）单设风险指标。本研究已有读数：
   固定 50 万下 25 年内破产 5.8%（POOL）；39 万下 3.4%。
2. **大学支出改条件预算**：yr15 时若组合低于护栏区间，大学预算从"美国"路径降级到"中国/其他"
   ——把今天的固定抽签变为届时的条件决策，比模型假设更可控（实务上用户天然会这么做）。
3. **Floor-aware 双层护栏**（基础生活 floor + 可调层，adj 只作用于可调层）：产品 backlog 候选。
4. **负债期限分段配置**：前 25 年现金流单独匹配低波动资产——方向与既有研究的
   "20/70/10 尾部稳健替代"（[[project-allocation-cew-decision]]）一致。
5. **阶段化 target**（35-60 保守、社保开始后放宽）：产品不支持 time-varying target，进阶思路。
