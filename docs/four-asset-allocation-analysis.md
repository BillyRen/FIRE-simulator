# 四类资产配置分析：加入房地产后的最优退休投资组合

> 分析日期：2026-03-13（数据修正后更新）
> 数据源：JST Macrohistory R6 + 2021-2025 扩展（16 国，1900-2025）
> 方法：Block Bootstrap Monte Carlo，GDP-sqrt 池化，2000 次模拟
> 数据修正：Housing_TR 使用乘法公式 `(1+capgain)*(1+rent_yd)-1`；
> Rent_Growth 从 filled 租金收益率系列计算，过渡行使用 inflation proxy

---

## 1. 为什么要加入房地产？

传统退休组合只有股票和债券两类资产。但 JST 数据集包含了 16 个发达国家超过 125 年的住宅房地产总回报（房价增值 + 租金收入），让我们可以评估房地产作为第四类资产的配置价值。

## 2. 四类资产的历史表现

基于 1900-2025 年、16 国的实际（扣除通胀后）回报统计：

| 资产 | 实际年均回报 | 年化波动率 | 夏普比率 |
|------|-----------|----------|---------|
| 国内股票 | 7.1% | 22.3% | 0.32 |
| 全球股票 | 9.4% | 31.9% | 0.29 |
| 债券 | 1.9% | 10.8% | 0.18 |
| **房地产（总回报）** | **6.9%** | **10.8%** | **0.64** |
| 房地产（仅资本增值） | 1.9% | 10.3% | 0.19 |

### 关键发现

**房地产是四类资产中夏普比率最高的**——回报接近股票（6.9% vs 7.1%），波动率只有股票的一半（10.8% vs 22.3%），夏普比率是股票的两倍（0.64 vs 0.32）。

这个优势完全来自**租金收入**。仅看房价增值，房地产的实际回报只有 1.9%（和债券一样）。租金收益率历史中位数约 4.6%，把房地产从一个平庸的资产提升为夏普比率最高的资产。

### 租金收益率的长期趋势

| 年代 | 中位租金收益率 |
|------|-------------|
| 1900s | 5.1% |
| 1950s | 4.7% |
| 1980s | 4.4% |
| 2000s | 3.8% |
| 2020s | 3.1% |

租金收益率在过去 120 年里持续下降（从 ~5% 降至 ~3%），这意味着未来的房地产总回报可能低于历史水平。但即使用 3% 的租金收益率，房地产的夏普比率仍然优于债券。

### 与其他资产的相关性

|  | 国内股 | 全球股 | 债券 | 房地产 |
|--|--------|--------|------|--------|
| 国内股 | 1.00 | 0.27 | 0.29 | **0.16** |
| 全球股 | 0.27 | 1.00 | 0.09 | **0.06** |
| 债券 | 0.29 | 0.09 | 1.00 | **0.18** |
| 房地产 | **0.16** | **0.06** | **0.18** | 1.00 |

房地产与股票、债券的相关性都很低（0.06-0.18），是组合中最好的分散化工具。特别是与全球股票的相关性仅 0.06，接近于不相关。

## 4. 费用率敏感性

以推荐配置 25/50/0/25 为例，测试房产费用率从 0.5% 到 3.0%：

| 房产费用率 | 成功率 | 中位终值 | P10 |
|---|---|---|---|
| 0.5% | 96.0% | 829M | 87M |
| **1.0%** | **95.8%** | **759M** | **73M** |
| 1.5% | 95.3% | 695M | 59M |
| 2.0% | 94.5% | 634M | 50M |
| 2.5% | 94.0% | 579M | 42M |
| 3.0% | 93.5% | 525M | 34M |

即使用最悲观的 3.0% 费用率（相当于直接持有物业的全部成本），成功率仍比当前纯股票配置高 2.9 个百分点。**分散化带来的尾部风险改善远大于费用的拖累。**

## 5. 费用率的合理取值

JST 的 `housing_tr` 是基于房价指数 + 租金收益率编制的"指数级"回报，不含任何交易或持有成本。实际投资需要扣除费用：

| 投资方式 | 建议费用率 | 构成 |
|---------|-----------|------|
| 房地产 ETF/REITs | 0.5-1.0% | 基金管理费 0.3-0.5% + 交易成本 |
| 直接持有（委托管理） | 1.5-2.5% | 维修 ~1% + 物业税/费 ~0.5% + 空置 + 管理费 |
| 直接持有（自管） | 1.0-1.5% | 维修 + 物业税/费，省管理费 |

**建议采用 1.0%**（假设通过房地产基金/REITs 投资）。

### 关于 REITs 的 caveat

通过公开交易的 REITs 投资房地产，其波动率会**显著高于**直接持有（REITs 与股市的相关性约 0.6-0.7）。JST 数据反映的是直接持有房产的回报特征（低波动、低相关），如果用 REITs 替代，分散化效果会打折。

理想的房地产敞口是**非上市房地产基金**或**直接持有**，但这些方式的流动性差、最低投资额高、管理成本更高。这是房地产配置的核心 trade-off。

## 6. 结论与建议

### 最优配置

| 风格 | 推荐配置 (国内/全球/债/房) | 成功率 | 核心逻辑 |
|------|--------------------------|--------|----------|
| **激进** | **25/50/0/25** | **95.8%** | 高成功率 + 尾部风险显著改善 |
| 更激进 | 20/40/0/40 | 96.6% | 最高成功率，但房产配比偏高 |
| 均衡 | 20/30/10/40 | 94.4% | 加入少量债券，波动率更低 |
| 稳健 | 10/20/20/50 | 91.5% | 房地产主导，最低波动 |

### 推荐配置详解：25% 国内股 / 50% 全球股 / 0% 债券 / 25% 房地产

相比当前的 33/67/0/0 纯股票配置：

- **成功率提升 5.2 个百分点**（90.6% → 95.8%）
- **P10（最差 10% 情况）从 4M 跃升至 73M**——尾部风险大幅改善
- **P25 从 185M 提升至 293M**
- 中位终值反而略有提升（725M → 759M）
- 不需要债券——房地产已提供比债券更好的分散化（更高回报 + 类似波动率 + 更低相关性）

### 为什么推荐 25% 而非 40%？

虽然 20/40/0/40 的成功率更高（96.6%），但 25% 是更务实的选择：

1. **流动性约束**：房地产（无论直接持有还是非上市基金）的流动性远低于股票，高配比会导致再平衡困难
2. **集中风险**：单一房产/单一市场的风险远高于指数级别的分散化
3. **边际递减**：从 0% 增加到 25% 的改善（成功率 +5.2%）远大于从 25% 增加到 40%（+0.8%）
4. **数据局限**：JST 指数反映的是全国性住宅市场，实际投资的波动可能更大

### 总结

房地产是一个被低估的退休资产配置选项。其核心价值不在于最大化回报（那是股票的工作），而在于以**接近债券的波动率**提供**接近股票的回报**，同时与其他资产**几乎不相关**。在退休组合中配置 25% 的房地产，可以在不牺牲中位数回报的情况下，显著改善最差情况的结果。

## 附录：数据修正说明

本次分析使用了修正后的 Housing_TR 数据。修正内容：

1. **乘法公式**：Housing_TR 回退重建从加法 `capgain + rent_yd` 改为乘法 `(1+capgain)*(1+rent_yd)-1`，与 JST 总回报定义一致
2. **Rent_Growth 来源**：从 filled 租金收益率系列计算（含 median-yield 回退行），而非原始缺失列
3. **过渡行处理**：imputed↔observed 边界行的 Rent_Growth 使用 inflation proxy，避免合成值产生的伪增长率

修正仅影响 ~9 个历史过渡行和 ~180 个 median-yield 回退行的 Rent_Growth 值。Housing_TR 与 JST 原始值的匹配率为 100%（1870/1870 行）。
