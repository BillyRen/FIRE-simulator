# 四类资产配置分析：加入房地产后的最优退休投资组合

> 分析日期：2026-03-13（数据修正后更新）
> 数据源：JST Macrohistory R6 + 2021-2025 扩展（16 国，1900-2025）
> 方法：Block Bootstrap Monte Carlo，GDP-sqrt 池化，2000 次模拟
> 房产费用率：1.5%（自住房产持有成本）
> 个人房产波动率：指数的 1.5 倍（加非系统性噪声建模）

---

## 1. 为什么要加入房地产？

传统退休组合只有股票和债券两类资产。但 JST 数据集包含了 16 个发达国家超过 125 年的住宅房地产总回报（房价增值 + 租金收入），让我们可以评估房地产作为第四类资产的配置价值。

## 2. 四类资产的历史表现

基于 1900-2025 年、16 国的实际（扣除通胀后）回报统计：

| 资产 | 实际年均回报 | 年化波动率 | 夏普比率 |
|------|-----------|----------|---------|
| 国内股票 | 7.1% | 22.3% | 0.32 |
| 全球股票 | 9.4% | 31.9% | 0.29 |
| 债券 | 1.9% | 10.8% | 0.18 |
| **房地产指数（总回报）** | **6.9%** | **11.7%** | **0.59** |
| 房地产指数（仅资本增值） | 1.9% | 10.3% | 0.19 |
| **个人房产（估算）** | **6.9%** | **17.6%** | **0.39** |

### 指数 vs 个人房产

JST 数据反映的是全国住宅市场指数，而个人房产集中于单一地段，承受额外的非系统性风险。研究表明个人房产的波动率约为指数的 1.5 倍。本分析对个人房产建模如下：

- 个人回报 = 指数回报 + ε，其中 ε ~ N(0, σ_idio)
- σ_idio = √(1.5²-1) × σ_index ≈ 13.1%
- 个人房产波动率 = √(11.7²+13.1²) ≈ 17.6%（指数的 1.5 倍）
- 与其他资产的相关性降至指数的 2/3（如与股票：0.16 → 0.11）

这意味着个人房产虽然均值回报不变，但风险更高、分散化效果打折。

### 租金收益率的长期趋势

| 年代 | 中位租金收益率 |
|------|-------------|
| 1900s | 5.1% |
| 1950s | 4.7% |
| 1980s | 4.4% |
| 2000s | 3.8% |
| 2020s | 3.1% |

### 与其他资产的相关性（指数级）

|  | 国内股 | 全球股 | 债券 | 房地产 |
|--|--------|--------|------|--------|
| 国内股 | 1.00 | 0.27 | 0.29 | **0.16** |
| 全球股 | 0.27 | 1.00 | 0.09 | **0.06** |
| 债券 | 0.29 | 0.09 | 1.00 | **0.18** |
| 房地产 | **0.16** | **0.06** | **0.18** | 1.00 |

个人房产的相关性约为上表的 2/3（因非系统性噪声稀释）。

## 3. Monte Carlo 模拟结果

所有结果均使用个人房产波动率（1.5x 指数），房产费用率 1.5%，金融资产费用率 0.5%。

## 4. 个人房产波动率的影响

比较指数级回报（JST 直接数据）与加入非系统性风险后的结果：

| 配置 | 指数波动率 | 1.5x 个人波动率 | 差异 |
|---|---|---|---|
| 25/50/0/25 | 95.3% | 95.0% | -0.3pp |
| 20/40/0/40 | 96.0% | 94.8% | -1.2pp |
| 17/33/0/50 | 95.8% | 93.9% | -1.9pp |

**房产配比越高，非系统性风险的拖累越大。** 这正是最优配比从指数级的 40% 下移到个人持有的 25-30% 的原因。

### 波动率倍数敏感性（20/40/0/40 配置）

| 波动率倍数 | 成功率 | P10 | P25 |
|-----------|--------|-----|-----|
| 1.0x（指数） | 96.0% | 75M | 264M |
| 1.25x | 96.0% | 72M | 226M |
| **1.5x（基准假设）** | **94.8%** | **53M** | **198M** |
| 1.75x | 93.6% | 36M | 164M |
| 2.0x | 92.5% | 19M | 136M |
| 2.5x | 88.3% | 0 | 85M |

即使个人房产波动率达到指数的 2 倍，40% 房产配置的成功率（92.5%）仍高于纯股票基准（90.6%）。2.5 倍时才跌破基准。

## 5. 费用率说明

本分析使用 **1.5% 费用率**，对应自住房产的年持有成本：

| 费用项 | 年化比例 |
|--------|---------|
| 维修保养 | ~1.0% |
| 物业费/税费 | ~0.5% |
| **合计** | **~1.5%** |

自住房产不产生空置损失和管理费，是成本最低的房地产持有方式。

### 关于 REITs

本分析不适用于公开交易的 REITs。REITs 与股市的相关性约 0.6-0.7，波动率也接近股票水平，与 JST 直接房产数据的低波动率（11.7%）、低相关性（0.06-0.18）特征截然不同。将本文的指数级结果（40% 最优）套用到 REITs 配置会严重高估分散化效果。

三种房地产投资方式的风险特征差异很大，不应混用：

| 投资方式 | 波动率 | 与股票相关性 | 本文是否建模 |
|---------|--------|------------|------------|
| 直接房产指数 | ~11.7% | 0.06-0.16 | 是（1.0x 场景） |
| 自住/单一房产 | ~17.6% | 0.04-0.11 | 是（1.5x 场景） |
| 上市 REITs | ~20-25% | 0.6-0.7 | **否** |

## 6. 结论

### 数据驱动的最优配置

考虑个人房产 1.5x 波动率后，最优配置为：

**25% 国内股 / 50% 全球股 / 0% 债券 / 25% 房地产**

（25-30% 房产区间内成功率差异仅 0.2pp，基本等效）

| 指标 | 基准 33/67/0/0 | 最优 25/50/0/25 | 变化 |
|------|---------------|----------------|------|
| 成功率 | 90.6% | 95.0% | **+4.4pp** |
| P10（最差 10%） | ¥4M | ¥55M | **+13.8x** |
| P25 | ¥185M | ¥232M | **+25%** |
| 中位终值 | ¥725M | ¥657M | -9.4% |

### 配比曲线

| 房产占比 | 成功率（1.5x vol） | 成功率（指数 vol） |
|---------|-------------------|-------------------|
| 0% | 90.6% | 90.6% |
| 10% | 93.2% | 92.8% |
| 20% | 94.5% | 94.2% |
| **25%** | **95.0%** | 95.3% |
| **30%** | **95.2%** | 95.6% |
| 40% | 94.8% | **96.0%** |
| 50% | 93.9% | 95.8% |
| 60% | 92.0% | 95.3% |

- **直接房产指数**（如非上市房产基金）：最优 40%，峰值平坦（30-50% 区间差异 <1pp）
- **自住/单一房产**：最优 25-30%，超过 40% 后显著递减

### 总结

房地产是一个被低估的退休资产配置选项。

- **直接房产指数**的核心优势是低波动率（11.7%，接近债券）和低相关性（0.06-0.18），最优配比 **40%**
- **自住房产**因集中于单一地段，波动率升至 17.6%（高于债券的 10.8%，但仍低于股票），分散化效果打折，最优配比 **25-30%**

两种情况下，在退休组合中加入房地产都能显著改善尾部风险（P10 从接近破产提升到安全水平），同时中位数回报仅小幅下降。但自住房产的投资者应注意，其承受的风险高于债券，收益约束来自单一地段的不可分散风险。

## 附录：数据修正说明

本次分析使用了修正后的 Housing_TR 数据。修正内容：

1. **乘法公式**：Housing_TR 回退重建从加法 `capgain + rent_yd` 改为乘法 `(1+capgain)*(1+rent_yd)-1`，与 JST 总回报定义一致
2. **Rent_Growth 来源**：从 filled 租金收益率系列计算（含 median-yield 回退行），而非原始缺失列
3. **过渡行处理**：imputation 边界行（imputed↔observed 双向）的 Rent_Growth 使用 inflation proxy，避免 yield 不连续性产生的伪增长率

修正仅影响 ~9 个历史边界行和 ~180 个 median-yield 回退行的 Rent_Growth 值。Housing_TR 与 JST 原始值的匹配率为 100%（1870/1870 行）。

## 附录：非系统性风险建模方法

个人房产回报 = 指数回报 + ε

- ε ~ N(0, σ_idio)，i.i.d.，与所有其他资产不相关
- σ_idio = √(k²-1) × σ_index，其中 k = 波动率倍数（默认 1.5）
- 效果：均值不变，波动率放大 k 倍，与其他资产的相关性缩小为 1/k

这比简单放大指数偏离（`mean + k × (return - mean)`）更合理：放大偏离会保持相关性不变，而实际上个人房产的非系统性风险应该降低与其他资产的相关性。
