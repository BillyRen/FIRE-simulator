# 四类资产配置分析：加入房地产后的最优退休投资组合

> 分析日期：2026-03-13
> 数据源：JST Macrohistory R6 + 2021-2025 扩展（16 国，1900-2025）
> 方法：Block Bootstrap Monte Carlo，GDP-sqrt 池化，2000 次模拟

---

## 1. 为什么要加入房地产？

传统退休组合只有股票和债券两类资产。但 JST 数据集包含了 16 个发达国家超过 125 年的住宅房地产总回报（房价增值 + 租金收入），让我们可以评估房地产作为第四类资产的配置价值。

## 2. 四类资产的历史表现

基于 1900-2025 年、16 国的实际（扣除通胀后）回报统计：

| 资产 | 实际年均回报 | 年化波动率 | 夏普比率 |
|------|-----------|----------|---------|
| 国内股票 | 7.1% | 21.9% | 0.32 |
| 全球股票 | 9.5% | 32.4% | 0.29 |
| 债券 | 1.9% | 10.8% | 0.18 |
| **房地产（总回报）** | **6.9%** | **10.7%** | **0.65** |
| 房地产（仅资本增值） | 1.9% | 10.3% | 0.19 |

### 关键发现

**房地产是四类资产中夏普比率最高的**——回报接近股票（6.9% vs 7.1%），波动率只有股票的一半（10.7% vs 21.9%），夏普比率是股票的两倍（0.65 vs 0.32）。

这个优势完全来自**租金收入**。仅看房价增值，房地产的实际回报只有 1.9%（和债券一样）。租金收益率历史中位数约 4.6%，把房地产从一个平庸的资产提升为夏普比率最高的资产。

### 租金收益率的长期趋势

| 年代 | 中位租金收益率 |
|------|-------------|
| 1900s | 5.2% |
| 1950s | 4.7% |
| 1980s | 4.4% |
| 2000s | 3.9% |
| 2020s | 3.1% |

租金收益率在过去 120 年里持续下降（从 ~5% 降至 ~3%），这意味着未来的房地产总回报可能低于历史水平。但即使用 3% 的租金收益率，房地产的夏普比率仍然优于债券。

### 与其他资产的相关性

|  | 国内股 | 全球股 | 债券 | 房地产 |
|--|--------|--------|------|--------|
| 国内股 | 1.00 | 0.27 | 0.29 | **0.16** |
| 全球股 | 0.27 | 1.00 | 0.09 | **0.06** |
| 债券 | 0.29 | 0.09 | 1.00 | **0.18** |
| 房地产 | **0.16** | **0.06** | **0.18** | 1.00 |

房地产与股票、债券的相关性都很低（0.06-0.18），是组合中最好的分散化工具。特别是与全球股票的相关性仅 0.06，接近于不相关。

## 4. 费用率敏感性

以推荐配置 25/50/0/25 为例，测试房产费用率从 0.5% 到 3.0%：

| 房产费用率 | 成功率 | 中位终值 | P10 |
|---|---|---|---|
| 0.5% | 95.3% | 670M | 66M |
| **1.0%** | **94.8%** | **611M** | **54M** |
| 1.5% | 93.9% | 555M | 44M |
| 2.0% | 93.4% | 505M | 34M |
| 2.5% | 93.0% | 462M | 25M |
| 3.0% | 92.3% | 418M | 17M |

即使用最悲观的 3.0% 费用率（相当于直接持有物业的全部成本），成功率仍比当前纯股票配置高 1.2 个百分点。**分散化带来的尾部风险改善远大于费用的拖累。**

## 5. 费用率的合理取值

JST 的 `housing_tr` 是基于房价指数 + 租金收益率编制的"指数级"回报，不含任何交易或持有成本。实际投资需要扣除费用：

| 投资方式 | 建议费用率 | 构成 |
|---------|-----------|------|
| 房地产 ETF/REITs | 0.5-1.0% | 基金管理费 0.3-0.5% + 交易成本 |
| 直接持有（委托管理） | 1.5-2.5% | 维修 ~1% + 物业税/费 ~0.5% + 空置 + 管理费 |
| 直接持有（自管） | 1.0-1.5% | 维修 + 物业税/费，省管理费 |

**建议采用 1.0%**（假设通过房地产基金/REITs 投资）。

### 关于 REITs 的 caveat

通过公开交易的 REITs 投资房地产，其波动率会**显著高于**直接持有（REITs 与股市的相关性约 0.6-0.7）。JST 数据反映的是直接持有房产的回报特征（低波动、低相关），如果用 REITs 替代，分散化效果会打折。

理想的房地产敞口是**非上市房地产基金**或**直接持有**，但这些方式的流动性差、最低投资额高、管理成本更高。这是房地产配置的核心 trade-off。

## 6. 结论与建议

### 最优配置

| 风格 | 推荐配置 (国内/全球/债/房) | 成功率 | 核心逻辑 |
|------|--------------------------|--------|----------|
| **激进** | **25/50/0/25** | **94.8%** | 最高成功率，尾部风险显著改善 |
| 均衡 | 20/30/10/40 | 90.9% | 波动率更低，适合心理承受力有限的投资者 |
| 稳健 | 10/20/20/50 | 85.9% | 房地产主导，最低波动 |

### 推荐配置详解：25% 国内股 / 50% 全球股 / 0% 债券 / 25% 房地产

相比当前的 33/67/0/0 纯股票配置：

- **成功率提升 3.7 个百分点**（91.1% → 94.8%）
- **P10（最差 10% 情况）从 7M 跃升至 54M**——尾部风险大幅改善
- **P25 从 167M 提升至 225M**
- 中位终值基本持平（647M → 611M），牺牲微小
- 不需要债券——房地产已提供比债券更好的分散化（更高回报 + 类似波动率 + 更低相关性）

### 为什么 25% 而不是更多？

虽然均值-方差优化建议 60-70% 房地产配置，但 25% 是更务实的选择：

1. **流动性约束**：房地产（无论直接持有还是非上市基金）的流动性远低于股票，高配比会导致再平衡困难
2. **集中风险**：单一房产/单一市场的风险远高于指数级别的分散化
3. **边际递减**：从 0% 增加到 25% 的改善（成功率 +3.7%）远大于从 25% 增加到 40%（+额外 -0.8%，实际反而下降）
4. **数据局限**：JST 指数反映的是全国性住宅市场，实际投资的波动可能更大

### 总结

房地产是一个被低估的退休资产配置选项。其核心价值不在于最大化回报（那是股票的工作），而在于以**接近债券的波动率**提供**接近股票的回报**，同时与其他资产**几乎不相关**。在退休组合中配置 25% 的房地产，可以在几乎不牺牲中位数回报的情况下，显著改善最差情况的结果。
