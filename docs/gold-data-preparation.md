# 黄金数据准备（底层数据，未接入产品）

> 状态：**仅底层数据准备**。已生成 `data/jst_gold.csv`，但尚未接入 simulator
> 引擎 / portfolio / 前端。本文记录数据来源、构造方法与局限，供日后决定是否
> 把黄金作为一个可选资产类别加入产品时参考。

## 1. 问题

JST 数据集（`data/jst_returns.csv`，16 国 1871–2025）的每一列都是**本币名义
收益**（引擎内部再用各国 `Inflation` 列转成实际收益）。黄金没有"本币收益"的
天然概念——它在单一全球市场以**美元**计价。要把黄金纳入与其他资产相同的口径，
必须先把美元金价用**汇率**换算成各国本币收益。

## 2. 关键发现：汇率已经现成

JST R6 原始数据自带 `xrusd` 列（**本币 / 美元**汇率），覆盖 1870–2020；项目的
2021–2025 扩展（`jst_extension_2021_2025.csv`）也带 `xrusd`。`build_dataset_from_jst.py`
早已用它把外国股票收益换算成投资者本币（`Global_Stock`）。因此黄金本币收益可以
**完全复用现成汇率**，只需补一条全球美元金价年度序列。

## 3. 构造公式

与现有 `Global_Stock` 的 FX 换算完全平行：

```
gold_local[t]          = gold_usd[t] * xrusd[t]
Gold_Nominal_Return[t] = gold_local[t] / gold_local[t-1] - 1
                       = (gold_usd[t]/gold_usd[t-1]) * (xrusd[t]/xrusd[t-1]) - 1
```

- 结果是**名义**本币收益，与其它列口径一致；引擎现有的"除以 `1+Inflation`"实际
  收益转换可统一套用。
- 美国 `xrusd ≡ 1.0`，该列退化为纯美元金价收益。

## 4. 数据来源

| 数据 | 来源 | 文件 |
|---|---|---|
| 美元金价（年均，USD/盎司，1833–2025） | World Bank Commodity Markets "Pink Sheet"（1960–今）+ Timothy Green 历史序列（1833–1959），由 National Mining Association 拼接；开源镜像 `github.com/datasets/gold-prices`（CC0） | `data/raw/gold_usd_annual.csv` |
| 汇率 `xrusd`（本币/美元） | JST Macrohistory Database R6（1870–2020）+ 项目 2021–2025 扩展 | `data/raw/JSTdatasetR6.xlsx`、`data/raw/jst_extension_2021_2025.csv` |

**金价历史分期（已在数据中体现）：**
- 1871–1933：古典金本位，金价≈固定（~$18.9–20.7）→ 名义收益≈0，实际收益由
  通胀/通缩决定。
- 1934–1971：布雷顿森林体系，美元官价 $35/盎司（市场年均值在 1968–1971 两级
  市场期略有偏离）。
- 1972–今：自由浮动，采用市场年均价。

**手工补值**：开源镜像缺 **1960** 年（Green 与 World Bank 拼接缝）。已按 NMA/Green
口径补 `1960 = 35.27`（介于 1959 的 $35.10 与 1961 的 $35.25 之间，布雷顿森林伦敦
年均价）。见 `data/raw/gold_usd_annual.csv`。

## 6. 验证结果（经济合理性）

- **结构**：2340 行与 `jst_returns.csv` 主键 **1:1 完全一致**；仅 2 个 NaN。
- **金本位期（USA ≤1932）**：名义年均 **+0.21% ≈ 0**，符合固定金价。
- **浮动期（USA 1972+）**：名义 **+8.56%/年**、实际 **+4.63%/年**、年波动 **21.5%**
  （1979 +78%、1981 −31% 等极值与史实吻合）。
- **超长期（USA 1871+）**：实际 **+1.32%/年** —— 经典"黄金长期大致保值"结论。
- **跨国一致性**：浮动期各国实际黄金收益与美国相关性 **0.80–0.85**、跨国标准差
  约 **5%**，确认黄金作为全球实物资产在各币种下行为相似，差异仅来自实际汇率
  （PPP）偏离。

## 7. 局限与注意事项

1. **DEU 1945–1946 = NaN**：德国占领期无有效汇率市场（JST `xrusd` 本身缺失），
   1948 才货币改革。此期间帝国马克计价的黄金收益在数学上不可定义，保留 NaN 是
   诚实做法。若日后接入引擎，需决定填补策略（如置 0 冻结，或排除该国早期）。
2. **年均价 vs 年末价**：金价用的是**年均价**（与 JST 多数序列一致），不是年末
   收盘价。这会平滑掉年内波动，**系统性低估**黄金的真实年度波动（与股票/房产列
   的同类口径一致，但要注意黄金年内波动比股票更剧烈）。
3. **金本位期名义收益≈0 的双刃**：1871–1971 黄金名义收益几乎为零，其实际收益
   完全由各国通胀/通缩驱动。Block bootstrap 抽样会保留这种"金本位长期零名义、
   浮动期高波动"的结构突变——这是真实历史特征，但意味着黄金的统计特性高度
   依赖采样窗口落在哪个货币体制。
4. **2025 为年均价**：2025 金价年均 ~$3442（当年大涨），属完整年份均值。
5. **xrusd 口径**：JST 的 `xrusd` 是名义汇率；欧元区国家在 1999/2002 前后的腿
   值转换已由 JST R6 处理（详见 `scripts/DATA_UPDATE_GUIDE.md`）。
6. **未做相关性/CMA 整合**：本步骤只产出历史收益序列，未把黄金纳入
   `data/cme/` 的前瞻 CMA 或相关矩阵。若接入产品，黄金与股/债/房的相关结构
   （尤其危机期的避险负相关）需另行处理。
