# REITs 长期是否回归房地产本质？(2026-06-13)

**问题**（用户）：REITs 短期可能被股市情绪/beta 主导，但它的底层资产几乎就是房地产，
长期它的风险收益属性会不会更接近房地产、而不是股票？

**简短回答**：方向上对，但我们 53 个年度观测的样本太短，无法把"长期收敛"统计上钉死。
证据是**分层**的——最干净的正面证据是 **REITs 领先房产**（lead-lag），以及接近的长期
CAGR；多周期相关性则混杂、噪声大。本质上 **REITs = 带杠杆 + 套股市外壳的（商业）房地产**，
保留一份直接持房没有的永久股票 beta + 杠杆，所以即使长期也不会完全收敛到 JST 住宅房产。

**数据**：实际年度回报 1972–2024（REIT = Nareit All Equity；Housing = JST USA `Housing_TR`；
Stocks = JST USA `Domestic_Stock`；JST USA CPI 平减）。脚本
`analysis/reit_horizon_convergence.py`。背景见 [reit-vs-jst-housing](reit-vs-jst-housing-2026-06-13.md)。

## 必须处理的三个方法论陷阱

1. **平滑假象**：JST 房产是评估/成交平滑的，短期波动和相关性被人为压低 → 用 **Geltner
   一阶去平滑**还原。
2. **平滑随周期"洗掉"**：长期累计回报受平滑影响小，所以长周期相关性上升**部分是机械的**
   （去平滑效应），不能全算作向基本面收敛。
3. **领先-滞后**：REITs 每日定价，会**领先**评估型房产指数 → 用 lead-lag 检验"REITs 提前
   price 同一资产"。

## 证据分层

### ① 平滑确实存在且很大
```
AR(1) 自相关：REIT −0.04   Housing +0.62   Stocks +0.01
```
房产 AR(1)=+0.62 是教科书级平滑信号（流动资产≈0）。原始 REIT↔Housing=0.29 被压低了。

**Geltner 去平滑（ρ=0.62）后**：

| | 原始 | 去平滑后 |
|---|---|---|
| Housing 波动 | 4.0% | **8.4%**（仍 << REIT 17.8%） |
| corr(REIT, Housing) | 0.29 | **0.41** |
| corr(Stocks, Housing) | 0.30 | **0.44** |

→ 去平滑让 REIT 和房产更相关，**但也同等抬高了股票↔房产**。在年度频率上，修正平滑后
REIT 并没有变得"比股票更像房地产"（0.41 ≈ 0.44）。

### ② 最干净的正面证据：REITs 领先房产
```
corr(REIT_t, Housing_{t+k})  （k>0 = REIT 领先）
  k=−3:−0.27  k=−2:−0.23  k=−1:−0.05    房产领先 REIT：≈0 或负
  k= 0:+0.29
  k=+1:+0.23  k=+2:+0.15  k=+3:+0.06    REIT 领先房产：正、衰减
```
**不对称很清楚**：今天的 REIT 回报正向预测未来 1–2 年房产回报，反过来不行。这是"REITs 给
同一底层房地产提前定价（流动、前瞻）"的签名——支持用户观点最强的证据。

### ③ 多周期相关性：混杂、噪声大（无法证实）

| h | n | corr(REIT,Stocks) | corr(REIT,Housing) | |
|---|---|---|---|---|
| 1 | 53 | 0.61 | 0.33 | |
| 5 | 10 | 0.58 | **0.65** | 非重叠：出现预测的反超 |
| 10 | 5 | 0.88 | 0.55 | n=5，无意义 |

重叠窗口（自相关严重）下两个相关性随 h 都下降、REIT 始终离股票更近。非重叠 h=5 支持假说但
n=10。**53 个年度观测撑不起干净的长周期结论。**

### ④ 长期收益水平：三者几乎相同
```
实际 CAGR：REIT 6.95%  ≈  Housing 6.34%  ≈  Stocks 6.75%
```
REIT 落在房产和股票之间——与"长期 REIT 收益反映房地产基本面"一致，但不区分两种假说。

## 综合判断

REITs 既非纯股票、也非纯房地产，而是**加杠杆 + 证券化的（主要商业）房地产**。三个结构性原因
让它**即使长期也不会完全收敛到 JST 住宅房产**：

1. **杠杆**（典型 LTV 30–50%）→ 放大收益与波动 → REIT 波动 17.8% vs 去平滑房产 8.4%。
2. **物业类型**：REITs 持商业（写字楼/零售/公寓/工业/数据中心/通信塔），JST 是住宅 →
   现金流驱动不同。
3. **证券化**：每日流动性 + 股市管道 → 实物房产从不会有的短期情绪 beta。

一句话：**长期看 REITs 是"带杠杆的房地产"，短期看是"戴着房地产面具的股票"——两者都对。**
用户的"长期回归房地产本质"方向正确（lead-lag + 接近的长期 CAGR 支持），但 REITs 永久保留
直接持房没有的股票 beta + 杠杆。

## 局限与更强的检验（未做）

- **样本**：仅 53 个年度观测是最大瓶颈，非重叠 h=10 只剩 5 个区块。
- **物业类型错配**：用了 JST **住宅**房产对标 REITs 的**商业**地产；文献标准对标是
  **NCREIF Property Index**（商业、季度、1978+），季度频率观测数翻 4 倍。
- **可选延伸**：NCREIF / Case-Shiller 季度数据、交易型指数（MIT/RCA，绕开评估平滑）、
  正式 cointegration（Engle-Granger）。
- 文献结论（更长/更高频国际面板 + 正确去平滑）一致指向长期收敛：Hoesli & Oikarinen
  (2012, *JIMF*) "Are REITs real estate?"。
