## 核心洞察

### 1. 安全提取率范围

在95%成功率标准下，5种场景的安全提取率范围为 **2.1% ~ 3.0%**。

- 最保守: FIRE-ALL-US (2.1%)
- 最乐观: JST-USA (3.0%)

### 2. 4%法则在65年退休期的适用性

- **JST-ALL-GDP√**: 78.9% ✗
- **JST-USA**: 85.7% △ (边缘)
- **FIRE-1970-高费**: 55.8% ✗
- **FIRE-1970-低费**: 69.2% ✗
- **FIRE-ALL-US**: 70.5% ✗

### 3. 费率影响

美股费率从0.5%提高到2.5%（+2.0pp），对成功率的影响:

- WR=3.5%: 成功率下降 12.1pp
- WR=4.0%: 成功率下降 13.4pp

95%安全提取率差异: 0.4pp（低费=2.6%, 高费=2.2%）

### 4. 数据源敏感性

各提取率下5场景成功率最大差距:

- WR=3.0%: 12.9pp（共识度中）
- WR=4.0%: 29.9pp（共识度低）
- WR=5.0%: 37.2pp（共识度低）
- 最大分歧点: WR=5.3% (差距37.5pp)

### 5. 可执行建议

1. **所有场景均达90%成功率的最高提取率**: 2.6%
   → 这是跨数据源的"共识安全提取率"，最具鲁棒性。

2. **所有场景均达80%成功率的最高提取率**: 3.1%
   → 适合风险承受能力较高的投资者。

3. **费率每增加1%，安全提取率约降低**: ~0.2pp
   → 选择低费率ETF（VTI 0.03%, VT 0.07%）可显著提高安全边际。

4. **对中国VT投资者的建议**: 使用JST-ALL-GDP√的结果作为基准（最保守、最多样），
   安全提取率3.0%~3.9%（80-90%成功率），
   配合风险护栏策略动态调整提取额，效果更佳。

### 6. 收敛与分歧区间

- **高度共识区间** (差距<5pp): 2.0% ~ 2.4%
  → 在此范围内，数据源选择对结论影响最小。
- **高度分歧区间** (差距>15pp): 3.2% ~ 7.0%
  → 在此范围内，数据源选择对结论影响显著，需谨慎。

### 9. 给中国VT投资者的分层建议

以下以年支出$100,000为例，计算所需退休资产规模。

#### 保守派（追求95%+成功率）
- 基准提取率: 2.1%（基于JST-ALL-GDP√，最保守场景）
- 必须配合风险护栏策略，在市场下行时削减提取
- 所需资产: $100,000 / 2.1% = **$4,761,905**
- $4M资产对应年支出: **$84,000/年**

#### 平衡派（接受80-90%成功率）
- 基准提取率: 2.6%~3.1%（所有场景均认可的范围）
- 建议使用风险护栏策略，可在乐观市况下适度提高消费
- 所需资产: **$3,225,806~$3,846,154**
- $4M资产对应年支出: **$104,000~$124,000/年**

#### 激进派（接受70%成功率）
- 基准提取率: 3.4%（最保守场景的70%安全线）
- 必须配合灵活的支出调整策略和备用收入来源
- 所需资产: **$2,941,176**
- $4M资产对应年支出: **$136,000/年**
