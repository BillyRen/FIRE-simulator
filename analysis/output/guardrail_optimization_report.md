# 最优风险护栏策略研究报告

**参数**: 65年退休期, 2000条MC路径, 初始资产$1,000,000, 基准提取$30,000/年(3%)
**配置**: 33%国内/67%全球股票, 费率0.5%/0.5%, Block Bootstrap块长5-15年

## 研究方法

### 搜索空间

共 7,740 个有效参数组合，搜索以下参数：

| 参数 | 搜索范围 |
|:---|:---|
| target_success | 50%, 60%, 70%, 75%, 80%, 85%, 90%, 95% |
| upper_guardrail | 90%, 95%, 99% |
| lower_guardrail | 10%, 20%, 30%, 40%, 50%, 60%, 70% |
| adjustment_pct | 5%, 10%, 25%, 50%, 75%, 100% |
| adjustment_mode | amount, success_rate |
| min_remaining_years | 1, 5, 10, 15, 20 |

### 四阶段评估

1. **FIRE-Hist**: 美国历史回测 (1871-2025)
2. **FIRE-MC**: 美国蒙特卡洛模拟
3. **FIRE-MC-1970**: 美国1970+数据, 高费率(国内2.5%)
4. **JST-MC**: JST多国池化蒙特卡洛模拟

### 综合评分: 加权几何均值 WGeo3

```
WGeo3 = (US_Std × MC1970 × JST_MC) ^ (1/3)

其中:
  US_Std = √(FIRE_Hist_score × FIRE_MC_score)  — 美国标准数据 (有效权重 1/3)
  MC1970 = FIRE_MC_1970_score                   — 1970后高费率 (有效权重 1/3)
  JST_MC = JST_MC_score                         — 多国池化数据 (有效权重 1/3)
```

### 单数据源评分公式

```
Score = safety × (0.25×median_util + 0.15×p10_util + 0.20×disc + 0.20×floor + 0.20×smooth) × 15
```

| 维度 | 权重 | 含义 |
|:---|:---:|:---|
| safety | 门控 | p10_funded² — 第10百分位资产存续比的平方 |
| median_util | 25% | 中位数年均消费/初始资产 |
| p10_util | 15% | P10年均消费/初始资产 (尾部保护) |
| disc | 20% | 时间折现消费 (2%年化折现) |
| floor | 20% | P10(最低年消费/初始年消费) (下行保护) |
| smooth | 20% | 1/(1+20×median_downside_cv) (消费平滑度) |

"资产耗尽"定义: 资产归零 **或** 年消费降至初始消费的50%以下 (消费地板)。

## 核心发现

### 1. upper_guardrail = 99% 是绝对共识

在所有 target_success 水平下，**所有4个数据源的Top-1策略均采用 upper_guardrail = 99%**。搜索空间中90%和95%的选项从未进入Top-50。

护栏策略的核心价值在于"在成功率偏离目标时才触发调整"。99%意味着几乎只在成功率下降时削减消费，而在市场大涨时保持高消费——这是一种**上行放开、下行保护**的非对称策略。

### 2. adjustment_pct 应保持在 5%~10% 的"微调"水平

`adjustment_pct` 是最敏感的参数:

| adjustment_pct | FIRE-MC 平均分 | JST-MC 平均分 |
|:---:|:---:|:---:|
| **5%** | 1.1037 | 1.4495 |
| **10%** | 1.1184 | 1.5011 |
| 25% | 1.0619 | 1.4839 |
| 50% | 0.9888 | 1.4062 |
| 75% | 0.9311 | 1.3312 |
| 100% | **0.1748** | **0.5063** |

`adjustment_pct = 100%`（一步到位调整）的得分断崖式下跌——**渐进式微调远优于激进调整**。100%调整会导致消费剧烈波动，严重伤害平滑度和下行保护。

### 3. min_remaining_years = 5 年是最优剩余年限

| min_remaining_years | FIRE-MC 平均分 | JST-MC 平均分 |
|:---:|:---:|:---:|
| **5** | **0.9852** | **1.4221** |
| 1 | 0.9175 | 1.3547 |
| 10 | 0.8898 | 1.2721 |
| 15 | 0.8558 | 1.2019 |
| 20 | 0.8340 | 1.1477 |

太短(1年)会在临近退休终点时过度调整；太长(10-20年)则过早停止护栏机制。5年取得最佳平衡。

### 4. adjustment_mode 存在清晰的"模式切换点"

| target_success | FIRE-MC Top-1 | JST-MC Top-1 | Top-10主导 |
|:---:|:---:|:---:|:---|
| 50%~75% | **amount** | **amount** | amount完全主导 (10/10) |
| 80%~85% | **amount** | **amount** | amount为主 (7-10/10) |
| 90% | **amount** | **success_rate** | 开始分化 |
| 95% | **success_rate** | **success_rate** | success_rate显著占优 |

- **amount模式**: 将提取额向"目标提取额"调整。适合中低target(初始提取率高)，直接控制金额变动幅度。
- **success_rate模式**: 将当前成功率向目标成功率调整。适合高target(≥90%)，安全边际充足时基于成功率的调整更精准。

### 5. lower_guardrail 随 target_success 系统性变化

| target_success | 最优 lower_guardrail |
|:---:|:---:|
| 50%~60% | 40%~50% |
| 70%~75% | 60%~70% |
| 80%~85% | 70% |
| 90% | 30%~70% (分歧较大) |
| 95% | **30%~50%** (反而下降) |

target=95%时lower反降至30%~50%：初始提取率已极低(2.37%~2.70%)，成功率几乎始终很高，下护栏很少被触发，低lower反而给策略留出更大自由度。

## 跨数据源一致性

### 高一致性区间: target_success ≥ 85%

| target_success | 共识度 | 证据 |
|:---:|:---:|:---|
| **95%** | **极高** | 4个数据源Top-5有3个交集策略; Top-1在所有数据源排名≤5 |
| **90%** | 高 | Top-1跨数据源排名6-41，参数差异较小 |
| **85%** | 较高 | FIRE-MC Top-1在JST-MC排名#2，反之亦然 |

### 低一致性区间: target_success ≤ 75%

| target_success | 共识度 | 证据 |
|:---:|:---:|:---|
| **80%** | 中等 | FIRE-MC-1970和JST-MC完全一致，但与FIRE-Hist有分歧 |
| **75%** | 较低 | FIRE-MC Top-1在FIRE-MC-1970排名#91 |
| **70%** | 低 | FIRE-Hist Top-1在其他数据源排名>200 |
| ≤60% | 很低 | FIRE-Hist最优策略在其他数据源排名>100 |

**结论**: target_success ≥ 85%的策略跨数据源高度稳健，推荐优先使用。

## 推荐策略

### 首推: 保守-平衡型 (target=85%)

**适用对象**: 希望兼顾安全性和消费水平的投资者

| 参数 | 值 |
|:---|:---|
| target_success | **85%** |
| upper_guardrail | **99%** |
| lower_guardrail | **70%** |
| adjustment_pct | **10%** |
| adjustment_mode | **amount** |
| min_remaining_years | **5** |

- 初始提取率: ~3.5% ($4M → $140K/年)
- 成功率: 91%(FIRE-MC) / 89%(JST-MC)
- 跨数据源稳定性: FIRE-MC #1 → JST-MC #2 (高度一致)
- 护栏增值: 比固定提取同成功率多提取+11%~16%

### 备选: 极保守型 (target=95%)

| 参数 | 值 |
|:---|:---|
| target_success | **95%** |
| upper_guardrail | **99%** |
| lower_guardrail | **30%** |
| adjustment_pct | **5%** |
| adjustment_mode | **success_rate** |
| min_remaining_years | **5** |

- 初始提取率: ~2.5% ($4M → $100K/年)
- 成功率: 92%~93%
- WGeo3综合得分: 1.3034 (全局第一)
- 跨数据源稳定性: 极高 (4个数据源Top-5均入选)
- 注意: 此策略下护栏增值为负，本质上是"极低提取率+极保守护栏"

## 结论

1. **护栏策略的"甜蜜点"在 target_success = 80%~85%**: 此区间护栏增值最大(+16%~23%的初始提取率提升)，且跨数据源一致性良好。

2. **参数选择的稳健规则**: upper=99%(永远不变), adj=5%~10%(微调为王), mr=5(中等剩余年限), lower和mode随target调整。

3. **渐进式调整远优于激进调整**: adjustment_pct=100%的得分是5%的1/6到1/3。"温水煮青蛙"式的消费微调是护栏策略成功的核心。

4. **target_success ≥ 85%的策略跨数据源高度稳健**: 无论用美国历史、美国MC、1970后高费率、还是JST多国池化数据，结论基本一致。

5. **对于 target_success ≤ 75%，不同数据源会给出显著不同的最优策略**: 应谨慎选择，或倾向于多个数据源的共识参数。
